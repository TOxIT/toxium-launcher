## Toxium Launcher

Лаунчер для быстрого запуска Minecraft с готовым каталогом игровых установок, управлением профилями, поддержкой скинов и автоматической подготовкой локального инстанса перед игрой.
<br clear="left">
Лаунчер создан для игроков, которым нужен простой сценарий: выбрать профиль, выбрать установку и нажать «Играть». Всё техническое — загрузка файлов, подготовка окружения, проверка состояния установки, обновления и запуск Minecraft — лаунчер берёт на себя приложение.

## Скачать

Последняя версия доступна на странице релизов:

**https://github.com/toxit/toxium-launcher/releases/latest**

Откройте страницу последнего релиза и скачайте установщик Toxium Launcher (ToxiumLauncher_x.x.x_x64-setup.exe) для вашей системы. Сейчас основной публичный установщик распространяется для Windows.

После установки запустите приложение, создайте профиль и выберите доступную игровую установку в каталоге.

## Возможности
- запускать Minecraft без ручной настройки папок, модов и параметров запуска;
- держать игровые установки отдельно друг от друга;
- быстро переключаться между профилями;
- играть с локальным никнеймом;
- пользоваться скинами в поддерживаемых установках;
- обновлять лаунчер и игровые установки из одного интерфейса;
- управлять памятью и Java без редактирования файлов вручную.

### Быстрый запуск

Главный экран показывает выбранную игровую установку, её состояние и основное действие:

- «Установить» — если установка ещё не подготовлена на устройстве;
- «Обновить» — если доступна новая версия;
- «Играть» — если всё готово;
- «Остановить игру» — если Minecraft уже запущен;
- «Скоро» — если запуск для этой установки пока недоступен.

Лаунчер запоминает выбранную установку и позволяет вернуться к ней при следующем запуске.

### Каталог игровых установок

В разделе «Сборки» доступна библиотека установок с карточками, статусами и подробной информацией. Для каждой установки отображаются версия Minecraft, загрузчик, версия релиза и текущее состояние на устройстве.

Лаунчер поддерживает локальное управление сборками:

- открыть папку установленной игры;
- удалить локальный инстанс с устройства;
- заново подготовить установку при следующем запуске;
- обновить устаревшую установку.

Удаление локальной сборки не удаляет его из каталога: установка остаётся доступной и может быть подготовлена повторно.

### Автоматическая подготовка игры

При первом запуске выбранной установки лаунчер сам создаёт рабочую структуру папок, загружает необходимые файлы и подготавливает запуск Minecraft.

Поддерживаются отдельные директории для игровых данных, модов, конфигов, ресурспаков, шейдерпаков, сохранений, скриншотов, логов и баг-репортов.

Если часть файлов отсутствует или повреждена, лаунчер может заново подготовить установку перед запуском.

### Профили игроков

При первом запуске нужно создать профиль. Доступны два сценария:

1. **Обычный профиль** — локальный никнейм без онлайн-авторизации.
2. **Облачный-профиль** — вход через Ely.by с поддержкой никнейма и скина.

Профили можно добавлять, выбирать, переименовывать и удалять. Никнейм облачного-профиля привязан к аккаунту и не меняется вручную внутри лаунчера.

### Скины

Для поддерживаемых игровых установок лаунчер может использовать систему скинов Ely.by. Если активен профиль и установка поддерживает интеграцию со скинами, скин применяется при запуске Minecraft.

В управлении профилями можно открыть каталог скинов Ely.by и обновить предпросмотр скина.

### Настройки запуска

В разделе настроек доступны параметры, которые напрямую влияют на запуск игры.

Можно настроить:

- объём оперативной памяти для Minecraft;
- путь к Java;
- автоматическое определение Java;
- проверку выбранной Java;
- режим применения Ely.by-скинов;
- автоматическую проверку обновлений лаунчера.

По умолчанию лаунчер выделяет Minecraft 4 GB RAM. Значение можно изменить вручную в пределах от 1 GB до 32 GB.

### Java

Лаунчер может попытаться найти Java автоматически или использовать путь, выбранный вручную. В настройках есть проверка Java executable, чтобы убедиться, что выбранный файл действительно запускается и похож на корректную Java Runtime.

Если Java не найдена автоматически, выберите `java.exe` или `java` вручную и нажмите «Тест».

### Обновления лаунчера

Лаунчер умеет проверять наличие новых версий. Если включена автоматическая проверка, на главном экране появится компактное уведомление о доступном обновлении.

Обновление можно:

- установить прямо из лаунчера, если автоустановка доступна;
- открыть страницу скачивания, если требуется ручное скачивание.

Ручная проверка обновлений доступна в настройках.

## Что нужно знать

- **Имеется проблема с доступом загрузчика NeoForge. Без доступа к официальному ресурсу загрузчика запустить сборки на этом загрузчике будет не возможно.**
- Для первого запуска выбранной игровой установки может потребоваться время: лаунчер скачивает и подготавливает необходимые файлы.
- Для загрузки каталога, обновлений и игровых файлов нужен доступ к интернету.
- Каждая игровая установка хранится как отдельный локальный инстанс.
- Если установка удалена с устройства, она остаётся в каталоге и может быть установлена снова.
- Для скинов нужен профиль Ely.by и поддержка скинов со стороны выбранной установки.
- Если игра не запускается, проверьте путь к Java и объём выделенной памяти в настройках.
- При нестабильном интернете загрузка файлов может завершиться ошибкой. В таком случае повторите действие позже.


## Минимальный сценарий запуска

1. Скачайте последнюю версию Toxium Launcher со страницы релизов.
2. Установите и откройте лаунчер.
3. Создайте профиль или войдите через Ely.by.
4. Выберите игровую установку на главном экране или в каталоге.
5. Нажмите «Установить» или «Играть».
6. Дождитесь подготовки файлов при первом запуске.

## Статусы установок

В интерфейсе используются статусы:

- **Не установлена** — локальный инстанс ещё не создан;
- **Устанавливается** — лаунчер подготавливает файлы;
- **Установлена** — установка готова к запуску;
- **Есть обновление** — доступна более свежая версия установки;
- **Обновляется** — выполняется обновление;
- **Запущена** — Minecraft уже работает;
- **Скоро** — запуск пока недоступен.

## Где хранятся данные

Лаунчер хранит игровые инстансы и служебные файлы в пользовательской директории приложения. Для установленной игровой установки можно открыть папку прямо из интерфейса кнопкой «Папка».

## Безопасность и приватность

Toxium Launcher хранит локальные настройки, профили и состояние установок на устройстве пользователя. Данные профиля используются для запуска Minecraft и, при выборе Ely.by-профиля, для применения поддерживаемой системы скинов.
